  • Project Description

    My project basically works similarly to a browserâ€™s (such as Google) "suggested search" function, but with respect to a given input data set (a transcript of a Gutenberg book). It will be a "smart suggester", suggesting the top (most frequent) occurrences of 2-word phrases that follow a given input search "seed" (a string of 2, 3, or 4 words).

  • Instructions

    1. Run the python script from my public_html folder, ie: mdoron-repa30@moe:~/public_html$ ./PythonFinalProject1.py
    2. Input a 2 word seed (that must appear in the source text (Jane Eyre))... eg "in the" has many permutations
    3. Close (click the 'X') the histogram when ready
    4. Repeat steps 2 and 3 with three- and four-word seeds, respectively, from the text (can look at histogram labels to see some possibilities)
    5. After closing the final histogram, navigate any browser to the webpage: http://moe.stuy.edu/~mdoron-repa30/finalWebsite.html
    6. Click on the different tabs to see the results!
